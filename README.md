# ChatGPT UI

- 支持部署在 vercel(10s 接口限制)，railway（500 小时限制） 等平台，需要设置环境变量：API_KEY
  或者在设置里填入
- 支持打包 electron,需要自己部署后端（本项目不提供）,只配置了 mac 段参数

功能

- 支持 gpt4 模型切换
- 支持多种对话设置
- 自适应夜间模式
  ...
